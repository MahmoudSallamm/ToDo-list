#  To-Do List API

A simple **RESTful API** built with **Node.js**, **Express**, and **MongoDB** that allows users to create, read, update, and delete to-do tasks.

---

##  Features

- Add a new task ✅  
- Get all tasks 📋  
- Update a task 📝  
- Delete a task ❌  
- MongoDB integration 🛢️  
- Clean code structure

---

##  Tech Stack

- Node.js  
- Express.js  
- MongoDB + Mongoose  
- Postman (API Testing)
